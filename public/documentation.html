My Fave Flix App Documentation

This application will enable users to search and find infrmation about their favorite movie including directors, cast, cinematography, and genres. Users will be able to create their own profile with their personal information. and gather and save their list data within their profile. Users will also be able to make their own lists of their favorite movies, directors, and movie genres.


Add a New User

Endpoint: /users

Method Type: POST

Description: This endpoint allows users to register with their own id. The endpoint returns a JSON object with 3 key values: a string for "id", a string for "Name", and an array for "Favorite Movies"

Query Parameters: None

Expected Request Data: JSON object with data about user. The "name" is required

Example:

{
"Name" : "Marlene"
}

Expected Response Data: JSON object with data about user. The generated ID is included

Example: {
"id": 1,
"name": "Marlene",
"favoriteMovies": []
}

Response Status Codes

201: Created

400: Bad request- names are required for users!


Updating Users

/users/:id

Method Type: POST

Description: This endpoint allows users to update their username. Endpoint returns a JSON object with few key values. For key "name" it returns a string, for "favorite movies" an array, and for an "id", an integer.

Query Parameters: :id

Expected request data: JSON object with data about user.

Required: :id

Example:

localhost:8080/users/1


{
"Username": "newUsername"
}

Expected response data: JSON object with data about user which includes,"id" with integer value, "Name", and "favorite movies" with an array.

Example:

{ "id": 1,
"name": "Brielle Updated Name",
"favoriteMovies": [] }

Response status codes:

200: OK-

400: Bad request- No such user!


Deleting Users

/users/:id

Method Type: DELETE

Description: This endpoint allows users remove their account by their Username.Endpoint returns a confirmation message that users account has been deleted.

Query Parameters: :id,

Expected request data: Required: :id

Example:

localhost:8080/users/1

Expected response data: A confirmation message that users account has been deleted.

Example:

{ "user 1 has been deleted" }

Response status codes:

200: OK- User id has been deleted!

400: Bad request- No such user!


Get All Movies

/movies

Method Type: GET

Description: This endpoint allows users to get whole list of movies. Endpoint returns an array with list of objects. Each object contains data about movies.

Query Parameters: None

Expected request data: Required: /movies

Example:

localhost:8080/movies

Expected response data: An array with list of movies. Each object contains data about movies. Every key is described with a value of a string.

Example:

{
Title: "Titanic",
Description: "A seventeen-year-old aristocrat falls in love with a kind but poor artist aboard the luxurious, ill-fated R.M.S. Titanic."
Genre: { Name: "Drama" },
Director: { Name: "James Cameron"},
ImageURL: "https://upload.wikimedia.org/wikipedia/en/1/18/Titanic_%281997_film%29_poster.png"
}

Search Movies by Title

/movies/:movieTitle

Method Type: GET

Description: This endpoint allows users to search a movie by title. Endpoint returns an object with data about specific movie.

Query Parameters: :movieTitle

Expected request data: Required: :movieTitle

Example:

localhost:8080/movies/Titanic

Expected response data: An object containing data about specific movie. Every key is described with a value in form of a string.

Example:

{
Title: "The Shawshank Redemption",
Description: "Two imprisoned men bond over a number of years, finding solace and eventual redemption through acts of common decency.",
Genre: { Name: "Drama" },
Director: { Name: "Frank Darabont" },
ImageURL: "https://www.imdb.com/title/tt0332280/mediaviewer/rm1153669376/?ref_=tt_ov_i" }

Response status codes:

200: OK- JSON object

400: Bad Request- No such movie!

Add Favorite Movies

/users/:id/movieTitle

Method Type: POST

Description:This endpoint allows users to add a movie to their favorites list.Endpoint returns a JSON object with users' data including an updated favorite movies array.

Query Parameters: :id, :movieTitle

Expected request data: Required: :id, :movieTitle

Example:

localhost:8080/users/1/Shawshank%20Redemption

Expected response data: JSON object with updated list of movies in Favorite Movies array.

{ Shawshank Redemption has been added to user 1 array! }

Response status codes:

200: OK- Movie has been added to user array!

400: Bad Request- No such user!


Delete Movie from Favorites

/users/:id/:movieTitle

Method Type: DELETE

Description: This endpoint allows users to remove movie from their list of favorites. Endpoint returns an JSON object with users data including updated favorite movies array.

Query Parameters: :id, :movieTitle

Expected request data: Required: /id, /movieTitle

Example:

localhost:8080/users/1/Thor

Expected response data: A message indicating the movie has been removed from the user's array.

Example:

{ Thor has been removed from user 1 array! }

Response status codes:

200: OK- Movie has been removed from users array!

400: Bad Request- No such user!
